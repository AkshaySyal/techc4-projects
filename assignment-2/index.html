Different tags in HTML

This page will describe effect and use of 5 different html tags

  1. Lists tag

    HTML lists allow web developers to group a set of related items in lists.

    An ordered list starts with the <ol> tag.

    Each list item starts with the <li> tag.

  2. Span tag

    The <span> element is an inline container used to mark up a part of a text, or a part of a document. Inline container means that the content inside it will remail in the same line

  3. Heading tags

    Heading tags are:

    • This is heading 1

    • This is heading 2

    • This is heading 3

    • This is heading 4

  4. Pragraph tag

    A paragraph <p> always starts on a new line, and is usually a block of text.

  5. Image tag

    Images can improve the design and the appearance of a web page.
